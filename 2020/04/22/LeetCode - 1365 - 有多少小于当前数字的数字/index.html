LeetCode - 1365 - 有多少小于当前数字的数字

Posted on 2020-04-22 | In 技术
Words count in article: 495 | Reading time ≈ 2

一、题目

  • 难度：简单
  • 题目链接：https://leetcode-cn.com/problems/how-many-numbers-are-smaller-than-the-current-number/
  • 语言： PHP
  • 题目内容：

    给你一个数组 nums，对于其中每个元素 nums[i]，请你统计数组中比它小的所有数字的数目。
    换而言之，对于每个 nums[i] 你必须计算出有效的 j 的数量，其中 j 满足 j != i 且 nums[j] < nums[i] 。
    以数组形式返回答案。

示例 1：
输入：nums = [8,1,2,2,3]
输出：[4,0,1,1,3]
解释：
对于 nums[0]=8 存在四个比它小的数字：（1，2，2 和 3）。
对于 nums[1]=1 不存在比它小的数字。
对于 nums[2]=2 存在一个比它小的数字：（1）。
对于 nums[3]=2 存在一个比它小的数字：（1）。
对于 nums[4]=3 存在三个比它小的数字：（1，2 和 2）。
示例 2：
输入：nums = [6,5,4,8]
输出：[2,1,0,3]
示例 3：
输入：nums = [7,7,7,7]
输出：[0,0,0,0]
提示：
2 <= nums.length <= 500
0 <= nums[i] <= 100

    思路：先进行正序排序，排序后数组下标就是比当前数字小的数目，但是需要避免重复数字时是否会影响结果。
